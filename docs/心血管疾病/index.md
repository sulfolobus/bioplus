# 心血管疾病

心血管疾病是一影响心脏和血管类系统的疾病，是全球范围内最主要的死亡原因之一。

## 主要类型

### 冠心病

冠状动脉粥样硬化导致心肌缺血缺氧。

### 高血压

血压持续升高，是心血管疾病的主要危险因素。

### 心力衰竭

心脏泵血功能下降，无法满足身体需求。

### 心律失常

心脏电活动异常，导致心跳过快、过慢或不规则。

### 脑血管疾病

包括缺血性和出血性卒中。

## 共同危险因素

| 因素 | 影响 |
|------|------|
| 高血压 | 最大危险因素 |
| 高血脂 | 动脉粥样硬化 |
| 糖尿病 | 血管损伤 |
| 吸烟 | 血管内皮损伤 |
| 肥胖 | 代谢综合征 |
| 缺乏运动 | 心血管功能下降 |
| 不良饮食 | 代谢异常 |

## 研究方向

### 精准医学

- 基因检测指导用药
- 个体化治疗方案

### 介入治疗

- 冠脉介入
- 结构性心脏病介入
- 电生理消融

### 再生医学

- 干细胞治疗
- 组织工程
