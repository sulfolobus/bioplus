# 冠心病治疗

冠心病是导致死亡的主要心血管疾病，本文介绍其治疗策略。

## 疾病特征

### 病因

- 冠状动脉粥样硬化
- 血管狭窄或阻塞
- 心肌供血不足

### 分型

| 类型 | 特点 |
|------|------|
| 稳定型心绞痛 | 症状稳定 |
| 不稳定型心绞痛 | 症状加重 |
| 心肌梗死 | 急性冠脉综合征 |

## 治疗策略

### 药物治疗

| 药物类型 | 代表药物 |
|----------|----------|
| 抗血小板 | 阿司匹林 |
| 他汀类 | 阿托伐他汀 |
| β受体阻滞剂 | 美托洛尔 |
| ACEI/ARB | 贝那普利 |

### 介入治疗

- PCI（经皮冠状动脉介入）
- 冠脉支架置入

### 外科治疗

- 冠脉搭桥术（CABG）
