# 高血压中西医治疗研究进展

高血压（Hypertension）是最常见的心血管疾病之一，分为原发性高血压和继发性高血压。

## 疾病定义

### 血压分类

| 分类 | 收缩压 (mmHg) | 舒张压 (mmHg) |
|------|---------------|---------------|
| 正常 | <120 | <80 |
| 正常高值 | 120-129 | <80 |
| 高血压1级 | 130-139 | 80-89 |
| 高血压2级 | ≥140 | ≥90 |
| 高血压危象 | ≥180 | ≥120 |

## 类型分类

### 原发性高血压

- 占 90-95% 
- 病因不明
- 多因素作用（遗传、环境、生活方式）
- 终身性疾病

### 继发性高血压

- 占 5-10%
- 有明确病因
- 常见原因：肾脏疾病、内分泌疾病、血管疾病
- 可能治愈

## 西医治疗

### 药物治疗

#### 一线药物

| 药物类别 | 代表药物 | 作用机制 |
|----------|----------|----------|
| 钙通道阻滞剂 | 氨氯地平 | 扩张血管 |
| ACE 抑制剂 | 贝那普利 | 抑制血管收缩 |
| ARB | 缬沙坦 | 阻断血管紧张素 |
| 利尿剂 | 氢氯噻嗪 | 减少血容量 |
| β受体阻滞剂 | 美托洛尔 | 降低心率 |

#### 新型药物

- **沙库比曲缬沙坦**：心衰和高血压双重获批
- **阿普昔腾坦**：难治性高血压新选择
- **Baxdrostat**：原发性醛固酮增多症新药
- **Aprocitentan**：难治性高血压新机制

### 非药物疗法

- 生活方式干预（DASH 饮食）
- 运动疗法
- 体重管理
- 戒烟限酒
- 压力管理

### 介入治疗

- **肾动脉去神经消融术**：难治性高血压新选择
- **颈动脉窦刺激器**：血压控制新方法

## 中医治疗

### 病因病机

| 证型 | 病机 | 症状 |
|------|------|------|
| 肝阳上亢 | 肝火上炎 | 头晕、面红 |
| 痰湿壅盛 | 痰浊内阻 | 肥胖、倦怠 |
| 阴虚阳亢 | 阴虚火旺 | 心悸、失眠 |
| 阴阳两虚 | 阴阳俱损 | 畏寒、乏力 |

### 治疗方法

#### 中药治疗

| 证型 | 代表方剂 |
|------|----------|
| 肝阳上亢 | 天麻钩藤饮 |
| 痰湿壅盛 | 半夏白术天麻汤 |
| 阴虚阳亢 | 杞菊地黄丸 |
| 阴阳两虚 | 金匮肾气丸 |

#### 中成药

- 松龄血脉康
- 珍菊降压片
- 牛黄降压丸
- 天麻钩藤颗粒

#### 非药物疗法

- 针灸治疗
- 推拿按摩
- 气功锻炼
- 耳穴贴压
- 中药足浴

### 中西医结合

#### 优势互补

1. **降压协同**：中西药联用增强疗效
2. **症状改善**：中药改善症状、提高生活质量
3. **减少副作用**：中药减轻西药不良反应
4. **靶器官保护**：中西医结合保护心脑肾

#### 典型方案

| 病情 | 方案 |
|------|------|
| 轻度高血压 | 中药+生活方式干预 |
| 中度高血压 | 西药+中药+生活方式 |
| 难治性高血压 | 介入+中西医结合 |

## 研究进展

### 精准医学

- 基因检测指导药物选择
- 药物基因组学研究
- 个体化治疗方案

### 新型药物研发

- **双受体激动剂**：GLP-1 受体/GLP-2 受体
- **新型利尿剂**：选择性 ENaC 抑制剂
- **RNA 干扰药物**：靶向降压

### 治疗策略

- **起始联合治疗**：早期联合获益更大
- **复方制剂**：提高依从性
- **血压目标值**：个体化制定

## 预防与管理

### 生活方式

| 因素 | 建议 |
|------|------|
| 钠盐摄入 | <5g/天 |
| 运动 | 每周 150 分钟 |
| 体重 | BMI 18.5-24.9 |
| 饮酒 | 男性<25g/天，女性<15g/天 |
| 饮食 | DASH 饮食、地中海饮食 |

### 自我监测

- 家庭血压监测
- 24 小时动态血压监测
- 智能设备辅助

### 长期管理

- 定期随访
- 依从性教育
- 并发症筛查
