# 心力衰竭治疗

心力衰竭是各种心脏疾病的终末阶段，本文介绍其治疗进展。

## 疾病特征

### 病因

| 类型 | 常见原因 |
|------|----------|
| 收缩性心衰 | 冠心病、心肌病 |
| 舒张性心衰 | 高血压、老年 |

### 分期

| 分期 | 特征 |
|------|------|
| A期 | 高危因素 |
| B期 | 结构改变 |
| C期 | 有症状 |
| D期 | 难治性心衰 |

## 治疗策略

### 药物治疗

| 药物类型 | 代表药物 |
|----------|----------|
| ACEI/ARB | 贝那普利 |
| β受体阻滞剂 | 美托洛尔 |
| 醛固酮拮抗剂 | 螺内酯 |
| SGLT2抑制剂 | 达格列净 |

### 器械治疗

- ICD（埋藏式除颤器）
- CRT（心脏再同步治疗）

### 外科治疗

- 心脏移植
- 人工心脏
