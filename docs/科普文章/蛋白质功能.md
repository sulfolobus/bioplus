# 蛋白质的功能

蛋白质是生命活动的主要执行者，由氨基酸组成，在生物体中承担着多种重要功能。

## 蛋白质的结构

### 一级结构

氨基酸序列决定了蛋白质的基本性质。

### 二级结构

蛋白质链折叠形成 α-螺旋和 β-折叠。

### 三级结构

蛋白质进一步折叠形成三维结构。

### 四级结构

多个蛋白质亚基组合形成功能性蛋白复合物。

## 蛋白质的主要功能

### 1. 酶催化

- 绝大多数酶是蛋白质
- 加速生物化学反应
- 例如：消化酶、DNA 聚合酶

### 2. 结构支持

- 胶原蛋白提供组织结构
- 角蛋白构成头发、指甲

### 3. 运输功能

- 血红蛋白运输氧气
- 载脂蛋白运输脂肪

### 4. 免疫防御

- 抗体识别并消灭病原体
- 干扰素抗病毒

### 5. 信号传递

- 胰岛素调节血糖
- 受体蛋白接收信号

### 6. 运动功能

- 肌动蛋白和肌球蛋白参与肌肉收缩
- 驱动蛋白参与细胞内运输

## 蛋白质与疾病

| 疾病 | 蛋白质问题 |
|------|------------|
| 阿尔茨海默病 | β-淀粉样蛋白异常 |
| 糖尿病 | 胰岛素分泌/作用异常 |
| 囊性纤维化 | CFTR 蛋白功能缺陷 |
