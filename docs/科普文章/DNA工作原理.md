# DNA 工作原理

DNA（脱氧核糖核酸）是生命体的遗传物质，承载着生物体的全部遗传信息。

## DNA 的结构

### 双螺旋结构

DNA 由两条反向平行的多核苷酸链组成，形成著名的双螺旋结构。

```
        A ==== T
       /      \
      T        A
       \      /
        G ==== C
       /      \
      C        G
       \      /
        A ==== T
```

### 碱基配对规则

- **腺嘌呤 (A)** 与 **胸腺嘧啶 (T)** 配对
- **鸟嘌呤 (G)** 与 **胞嘧啶 (C)** 配对

## DNA 的功能

### 1. 存储遗传信息

DNA 序列存储了构建和维持生命所需的全部信息。

### 2. 复制

DNA 在细胞分裂前进行精确复制，确保遗传信息传递给子细胞。

### 3. 转录

DNA 上的遗传信息被转录成 mRNA（信使 RNA）。

### 4. 翻译

mRNA 上的遗传密码被翻译成蛋白质序列。

## DNA 复制过程

1. **解旋**：解旋酶打开 DNA 双螺旋
2. **引物合成**：引物酶合成 RNA 引物
3. **延伸**：DNA 聚合酶沿模板链合成新链
4. **连接**：DNA 连接酶连接片段

## DNA 与基因

- **基因**是 DNA 上的功能片段
- 一个 DNA 分子包含多个基因
- 人类基因组约有 30 亿个碱基对

## DNA 技术应用

### 基因测序

读取 DNA 序列，用于疾病诊断、进化研究等。

### DNA 鉴定

用于法医鉴定、亲子鉴定等。

### 基因工程

改造 DNA 用于药物研发、转基因作物等。
