# 癌症是如何发生的？

癌症是一类细胞异常增殖并可能扩散到身体其他部位的疾病总称。

## 癌症的本质

### 正常细胞 vs 癌细胞

| 特征 | 正常细胞 | 癌细胞 |
|------|----------|--------|
| 生长方式 | 受控 | 不受控 |
| 增殖速度 | 适度 | 快速 |
| 细胞寿命 | 有限 | 无限 |
| 扩散能力 | 无 | 可能转移 |
| 基因稳定性 | 稳定 | 不稳定 |

## 癌症发生的原因

### 基因突变

癌症的根本原因是基因突变，导致细胞生长失去控制。

#### 突变类型

1. **原癌基因激活**
   - 促进细胞生长的基因过度活跃
   - 示例：RAS、MYC

2. **抑癌基因失活**
   - 抑制生长的基因失去功能
   - 示例：p53、RB

3. **修复基因缺陷**
   - DNA 修复机制失效
   - 示例：BRCA1/2

### 致癌因素

#### 外因

| 因素 | 致癌机制 |
|------|----------|
| 烟草 | 诱发基因突变 |
| 紫外线 | DNA 损伤 |
| 病毒 | HPV→宫颈癌，HBV→肝癌 |
| 化学物质 | 苯、亚硝胺等 |
| 辐射 | X 射线、氡气 |

#### 内因

- 遗传易感性
- 免疫系统功能
- 激素水平
- 慢性炎症

## 癌症发展过程

### 阶段

```
正常细胞 → 基因突变 → 癌前病变 → 原位癌 → 侵袭癌 → 转移
```

### 重要特征

1. **持续增殖信号**
2. **逃避生长抑制**
3. **抵抗细胞死亡**
4. **无限复制**
5. **血管生成**
6. **组织侵袭和转移**

## 常见癌症类型

| 癌症 | 常见原因 | 早期症状 |
|------|----------|----------|
| 肺癌 | 吸烟、空气污染 | 持续咳嗽 |
| 乳腺癌 | 遗传、激素 | 肿块 |
| 结直肠癌 | 饮食、遗传 | 便血 |
| 肝癌 | 肝炎、酒精 | 右上腹痛 |
| 胃癌 | 幽门螺杆菌 | 消化不良 |

## 癌症治疗

### 传统治疗方法

1. **手术**：切除肿瘤组织
2. **化疗**：使用药物杀死癌细胞
3. **放疗**：使用射线杀死癌细胞

### 现代治疗方法

| 方法 | 原理 | 优点 |
|------|------|------|
| 靶向治疗 | 针对特定基因突变 | 副作用小 |
| 免疫检查点抑制剂 | 激活免疫系统 | 持久效果 |
| CAR-T 细胞疗法 | 基因工程 T 细胞 | 个性化治疗 |
| 质子治疗 | 精准放疗 | 减少损伤 |

### 治疗效果

- **早期发现**：治愈率高
- **中期**：综合治疗为主
- **晚期**：延长生存，改善生活质量

## 预防策略

### 一级预防

- 戒烟限酒
- 健康饮食
- 适度运动
- 避免致癌物

### 二级预防

- 定期体检
- 癌症筛查
- 早发现早治疗

### 三级预防

- 规范治疗
- 康复管理
- 提高生活质量
