# 免疫系统是如何工作的？

免疫系统是人体抵御病原体入侵的防御系统，保护我们免受病毒、细菌和其他有害物质的侵害。

## 免疫系统的组成

### 免疫器官

| 器官 | 功能 |
|------|------|
| 骨髓 | 产生血细胞和免疫细胞 |
| 胸腺 | T 细胞成熟 |
| 淋巴结 | 过滤淋巴液，免疫细胞聚集 |
| 脾脏 | 过滤血液，储存免疫细胞 |
| 扁桃体 | 呼吸道入口的第一道防线 |

### 免疫细胞

- **白细胞**：免疫系统的主要执行者
- **巨噬细胞**：吞噬病原体
- **树突状细胞**：呈递抗原给 T 细胞
- **T 细胞**：细胞免疫
- **B 细胞**：体液免疫，产生抗体
- **自然杀伤细胞**：直接杀死被感染细胞

## 免疫机制

### 第一道防线：先天免疫

1. **物理屏障**
   - 皮肤
   - 黏膜
   - 呼吸道纤毛

2. **化学屏障**
   - 胃酸
   - 酶
   - 抗菌肽

3. **细胞防御**
   - 巨噬细胞吞噬
   - 自然杀伤细胞
   - 炎症反应

### 第二道防线：适应性免疫

#### T 细胞免疫

- **辅助 T 细胞**：协调免疫反应
- **细胞毒性 T 细胞**：直接杀死被感染细胞
- **记忆 T 细胞**：记住病原体

#### B 细胞免疫

- B 细胞产生抗体
- 抗体中和病原体
- 记忆 B 细胞长期保留

## 免疫反应过程

```
病原体入侵 → 抗原呈递 → T 细胞激活 → B 细胞产生抗体 → 抗体中和病原体 → 记忆细胞形成
```

## 免疫系统疾病

### 免疫缺陷

- **原发性**：遗传因素，如严重联合免疫缺陷病（SCID）
- **继发性**：HIV 感染导致艾滋病

### 自身免疫病

免疫系统错误攻击自身组织：

| 疾病 | 攻击目标 |
|------|----------|
| 类风湿关节炎 | 关节 |
| 系统性红斑狼疮 | 多器官 |
| 1型糖尿病 | 胰岛细胞 |
| 多发性硬化症 | 神经髓鞘 |

### 过敏

对外界无害物质的过度反应：

- 花粉过敏
- 食物过敏
- 药物过敏

## 增强免疫力的方法

### 健康生活方式

- 均衡饮食
- 适度运动
- 充足睡眠
- 减少压力

### 疫苗接种

疫苗帮助免疫系统"记住"病原体，产生长期保护。
