# 进化与分类学

本专题把生物多样性、系统发育和适应性进化联系起来，是理解“为什么会有这么多物种”的核心。

## 知识点清单

- 自然选择、人工选择与性选择
- 适应、适合度与生存压力
- 物种形成、隔离和辐射进化
- 分类系统与系统发育树
- 形态学、分子证据和同源/同功结构

## 典题

题目：两种外形相似但亲缘关系很远的动物，为什么不能仅凭形态进行分类？

解题思路：从同功结构、趋同进化和分子证据角度说明。

## 易错点

- 把进化理解成“朝更高级方向前进”
- 混淆同源结构和同功结构
- 用单一形态证据下分类结论
- 把种群变化和个体变化混为一谈

## 自测题

1. 为什么自然选择作用对象是表型而进化单位是种群？
2. 生殖隔离如何促进新物种形成？
3. 分子证据为什么能辅助分类？
4. 适应性辐射说明了什么进化现象？
