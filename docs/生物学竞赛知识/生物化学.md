# 生物化学

本专题关注生物分子的结构、能量转换、代谢通路和酶学原理，是竞赛里最需要理解与推演的部分。

## 知识点清单

- 氨基酸、蛋白质与酶的结构功能关系
- 糖类、脂质、核酸的化学基础
- ATP、NADH、FADH2 和能量货币概念
- 细胞呼吸、光合作用和代谢耦联
- 酶动力学、抑制剂、变性与调控
- 代谢通路的反馈调节和限速步骤

## 典题

题目：为什么温度升高在一定范围内会加快酶促反应，但过高又会使反应速率下降？

解题思路：先说明温度对分子碰撞频率的促进作用，再解释蛋白质空间结构破坏导致活性中心失活。

## 易错点

- 把 ATP 当成长期储能物质
- 忽略酶活性与蛋白质结构之间的关系
- 只背反应式，不理解代谢通路中的物质和能量流动
- 不会区分底物浓度、酶浓度、温度、pH 的作用机制
