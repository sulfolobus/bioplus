# 植物生理学

本专题聚焦植物的营养、运输、生长、发育和逆境响应，是竞赛中常见的综合推理模块。

## 知识点清单

- 水分吸收、矿质吸收和运输
- 光合作用与同化物分配
- 植物激素及其互作
- 向光性、向重力性、顶端优势
- 开花、果实发育与种子萌发
- 逆境响应与抗性机制

## 典题

题目：为什么去顶后侧芽生长增强，而喷施生长素后这一现象会被部分抑制？

解题思路：从顶端优势、生长素极性运输和浓度效应解释。

## 易错点

- 把生长素“促进生长”绝对化
- 不区分蒸腾作用和根压
- 忽略环境因子与激素作用的联动
- 认为所有植物激素都只在一个部位合成

## 自测题

1. 蒸腾作用为什么能促进水分运输？
2. 植物激素之间为什么常存在协同与拮抗？
3. 光周期现象与开花之间有什么关系？
4. 植物如何应对干旱或盐胁迫？
