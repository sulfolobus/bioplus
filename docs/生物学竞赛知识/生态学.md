# 生态学

本专题要求把生物与环境视为动态系统，重点考查群落结构、种群动态、能量流动和生态建模。

## 知识点清单

- 种群增长模型与密度制约
- 群落结构、物种关系和演替
- 生态系统组成、食物网和营养级
- 能量流动、物质循环和生态效率
- 生态位、边缘效应和生物多样性
- 生态恢复、保护与管理

## 典题

题目：某湖泊富营养化后水华频繁出现，可能涉及哪些生态机制？

解题思路：从营养盐输入、生产者暴发、溶氧下降、群落失衡和反馈回路分析。

## 易错点

- 把“生产者多”简单等同“生态系统更稳定”
- 混淆生态系统开放性与物质循环概念
- 只会背营养级，不会做能量计算
- 忽略时间尺度和空间尺度

4. 生物多样性为何与生态稳定性相关？
