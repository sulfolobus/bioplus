# 数据分析与科研表达

本专题训练竞赛中的图表分析、实验推理、科学写作与口头表达，是从“会做题”走向“拿高分”的最后一公里。

## 知识点清单

- 坐标图、柱状图、折线图和散点图判读
- 均值、误差、极值和趋势分析
- 相关性、因果性与假设检验基础
- 科学问题、假设、变量和结论表达
- 论证结构、实验报告和答题规范

## 典题

题目：某实验中两组曲线在 12 小时后出现明显分化，如何判断差异是否由实验处理造成？

解题思路：从对照设计、统计显著性、变量控制和机制解释四层判断。

## 易错点

- 把相关性误说成因果性
- 看趋势不看样本量和误差条
- 结论语言过于绝对
- 答题时只写现象，不写机制

## 自测题

1. 误差条通常能告诉我们什么信息？
2. 如何区分“支持假设”与“证明假设”这两个表述？
3. 一份高质量实验结论应包含哪些元素？
4. 面对未知数据，你如何组织推理顺序？
