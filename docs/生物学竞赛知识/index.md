# 生物学竞赛知识

这里面向准备生物学竞赛的学习者，聚焦更系统的知识整合、思维训练和进阶阅读。

## 竞赛重点

- 细胞生物学
- 分子遗传学
- 动植物生理
- 生态学与进化
- 生物化学基础
- 实验设计与读图能力

## 训练建议

### 1. 以专题为单位复习

不要只按教材顺序推进，可以按细胞、遗传、生态、实验等专题建立独立笔记。

### 2. 强化信息整合

竞赛题常常同时考察概念、图表和实验结果，需要训练快速提取关键信息的能力。

### 3. 关注科学表达

除了做题，还要练习用规范语言解释现象、写出假设并说明证据链。
