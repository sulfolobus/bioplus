## 学习路径

### 第 1 阶段：打地基

先学细胞生物学、生物化学和分子遗传学，建立最基础的理解框架。

### 第 2 阶段：建立系统

再学动物生理、植物生理和生态学，把“个体—种群—群落—生态系统”串起来。

### 第 3 阶段：冲击高分

最后用进化与分类学、生物技术与实验方法、数据分析与科研表达训练综合题和论述题。

## 训练建议

1. 每章都做“概念卡片 + 题目 + 错题复盘”。
2. 每周至少做一次图表题和一次实验设计题。
3. 所有论述题都坚持“结论—证据—机制—边界”四步法。

## 使用说明

- 本专题内容以竞赛常见考点为骨架，结合高中生物教学内容做系统扩展。
- 学习目标应尽量逼近“全面掌握”，但实际备赛仍需结合真题持续迭代。
