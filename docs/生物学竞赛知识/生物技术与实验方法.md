# 生物技术与实验方法

本专题面向竞赛中的实验设计、分子检测、操作规范和结果判读，是把知识转化为得分能力的关键。

## 知识点清单

- 显微镜、离心、移液和无菌操作
- PCR、凝胶电泳、测序和分子探针
- 酶联免疫、杂交、标记和示踪
- 实验设计、对照设置和误差分析
- 数据读图、统计和结果表达

## 典题

题目：如何通过凝胶电泳判断 PCR 扩增是否成功？

解题思路：结合条带位置、大小标准、阴阳性对照和可能的污染问题分析。

## 易错点

- 只关注结果图，不关注对照组
- 把“实验失败”简单归因于操作粗心
- 不会分析非特异条带和空白条带
- 统计题只报平均值，不看离散程度

## 自测题

1. 为什么实验中必须设置阴性和阳性对照？
2. PCR 反应体系中每种成分的作用是什么？
3. 如何判断一组实验数据是否支持假设？
4. 分子实验为什么特别强调污染控制？
