# 动物生理学

本专题集中考查动物个体如何维持内稳态、完成运动、感知、调节和免疫防御。

## 知识点清单

- 神经系统结构与兴奋传导
- 内分泌系统与激素反馈
- 循环、呼吸、消化和排泄系统协调
- 免疫系统组成、免疫应答和免疫记忆
- 运动系统、感觉器官和行为调节
- 体温调节、渗透调节和酸碱平衡

## 典题

题目：在剧烈运动后，人体为什么会出现呼吸频率升高、心率加快和乳酸暂时积累？

解题思路：从能量需求增加、缺氧代谢、体液调节和代偿反应四方面解释。

## 易错点

- 把神经调节和体液调节割裂理解
- 免疫题中忽视“先天免疫—适应性免疫”的递进关系
- 把激素作用理解成“直接改变细胞类型”
- 生理稳态题不看反馈环路
