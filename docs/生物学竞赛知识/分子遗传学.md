# 分子遗传学

本专题连接“遗传规律”和“分子机制”，是竞赛中最能拉开差距的核心模块。

## 知识点清单

- DNA、RNA 与遗传信息流动
- 复制、转录、翻译与基因表达调控
- 基因突变、重组、重排与遗传变异
- 孟德尔定律、连锁互换和系谱分析
- 伴性遗传、数量性状和多基因遗传
- 分子标记、基因测序和基因编辑基础

## 典题

题目：某突变导致蛋白质提前终止，但mRNA长度未变。最可能是哪一类突变？请说明。

解题思路：从密码子变化、无义突变、阅读框和蛋白长度变化角度分析。

## 易错点

- 只会算孟德尔比例，不会解释基因层面机制
- 把“突变”与“重组”混淆
- 伴性遗传判断时忽略父母和子代性别信息
- 认为基因表达只在“转录+翻译”两步完成

## 自测题

1. 为什么说 DNA 复制具有半保留特征？
2. 转录和翻译分别发生在哪里？
3. 系谱图中怎样识别常染色体显性遗传？
4. 基因调控为什么会影响表型差异？
