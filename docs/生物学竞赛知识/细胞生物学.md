# 细胞生物学

本专题聚焦细胞的结构、功能、分裂、代谢和显微分析，是竞赛里最基础也最常考的模块之一。

## 知识点清单

- 细胞学说与细胞的基本特征
- 原核细胞、真核细胞与特殊细胞结构
- 细胞膜、细胞器、细胞核与膜系统
- 跨膜运输、胞吞胞吐与细胞稳态
- 细胞周期、有丝分裂、减数分裂基础
- 显微观察、图像判读和实验设计

## 典题

题目：某细胞既能进行光合作用，又能进行有氧呼吸。根据结构特征判断其可能属于哪类细胞，并说明理由。

解题思路：从叶绿体、线粒体、细胞核、细胞壁等特征综合判断，同时注意“结构存在”与“功能实现”之间的关系。

## 易错点

- 将“原核”与“无细胞结构”混淆
- 把细胞器功能背成孤立结论，不理解协同关系
- 有丝分裂和减数分裂图像判读时不看染色体状态
- 把实验现象直接当作结论，忽略变量控制
