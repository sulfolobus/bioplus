# PCR检测技术

聚合酶链式反应（PCR）是分子诊断的基础技术，用于检测病原体和基因突变。

## 技术原理

### 基本原理

```
变性 → 退火 → 延伸 → 循环
```

**关键组件：**
- DNA聚合酶
- 引物
- dNTP
- 模板DNA

## 技术类型

| 类型 | 特点 | 应用 |
|------|------|------|
| 常规PCR | 定性 | 基因检测 |
| qPCR | 定量 | 病毒载量 |
| ddPCR | 高精度绝对定量 | 稀有突变 |

## 临床应用

### 感染性疾病

| 疾病 | 检测靶标 |
|------|----------|
| COVID-19 | SARS-CoV-2 RNA |
| 乙肝 | HBV DNA |
| 结核 | TB DNA |

### 肿瘤诊断

| 应用 | 说明 |
|------|------|
| 基因突变检测 | EGFR、KRAS |
| 融合基因检测 | ALK、ROS1 |
| 拷贝数变异 | HER2 |

## 分子诊断流程

1. 样本采集
2. DNA/RNA提取
3. PCR扩增
4. 结果分析
