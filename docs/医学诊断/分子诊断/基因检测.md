# 基因检测

基因检测是预防医学的重要组成部分，用于评估疾病风险和指导健康管理。

## 检测类型

### 携带者筛查

| 疾病 | 适用人群 |
|------|----------|
| 囊性纤维化 | 育前夫妇 |
| 脊髓性肌萎缩 | 高危人群 |
| 地中海贫血 | 高发地区 |

### 遗传风险评估

- 多基因风险评分
- 家族史分析
- 生活方式评估

## 检测技术

| 技术 | 特点 |
|------|------|
| 基因芯片 | 多病种筛查 |
| 全外显子测序 | 全面分析 |
| 全基因组测序 | 最全面 |

## 临床应用

### 精准预防

- 早筛早诊
- 个体化体检
- 健康管理

### 用药指导

- 药物基因组学
- 疗效预测
- 不良反应预防
