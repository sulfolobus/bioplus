# 医学影像AI

人工智能在医学影像分析中的应用最为成熟，已进入临床实用阶段。

## 应用领域

### X光和CT

| 应用 | 功能 |
|------|------|
| 肺结节检测 | 早期肺癌筛查 |
| 骨折检测 | 快速筛查 |
| 肺炎诊断 | COVID-19辅助 |

### MRI

| 应用 | 功能 |
|------|------|
| 脑肿瘤分割 | 精准定位 |
| 阿尔茨海默早筛 | 脑区分析 |
| 心脏成像 | 功能评估 |

### 超声

| 应用 | 功能 |
|------|------|
| 甲状腺结节 | 良恶性判断 |
| 乳腺病变 | BI-RADS分级 |
| 胎儿检查 | 生长发育评估 |

## 主流产品

| 产品 | 公司 | 适应症 |
|------|------|--------|
| Lunit | Lunit | 乳腺癌 |

- 数据不平衡
- 域适应问题
- 临床验证
