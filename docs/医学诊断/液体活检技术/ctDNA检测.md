# ctDNA 检测技术

循环肿瘤 DNA（ctDNA）检测是液体活检的核心技术，用于癌症的早筛和监测。

## ctDNA 生物学特征

### 来源

- 肿瘤细胞凋亡
- 肿瘤细胞坏死
- 循环肿瘤细胞裂解

### 特征

| 特征 | 说明 |
|------|------|
| 片段大小 | 约 180bp |
| 丰度 | 0.01%-10% |
| 半衰期 | 约 2小时 |

## 检测技术

### 数字 PCR（dPCR）

| 特点 | 说明 |
|------|------|
| 灵敏度 | 0.01% |
| 绝对定量 | 是 |
| 通量 | 较低 |

### NGS 测序

| 特点 | 说明 |
|------|------|
| 灵敏度 | 0.1% |
| 多重检测 | 是 |
| 发现新突变 | 是 |

### CAPP-Seq

| 特点 | 说明 |
|------|------|
| 灵敏度 | 0.02% |
| 靶向 | 是 |
| 成本 | 中等 |

## 临床应用

### 伴随诊断

- EGFR突变检测
-ALK融合检测
- ROS1融合检测

### 疗效监测

- 治疗反应评估
- 耐药突变检测
- 复发预警
