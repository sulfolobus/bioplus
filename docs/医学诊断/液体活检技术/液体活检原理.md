# 液体活检技术原理

液体活检是一种通过检测体液中的生物标志物来诊断疾病的技术，具有无创、便捷的优势。

## 技术原理

### 检测标志物

| 标志物 | 来源 | 应用 |
|--------|------|------|
| 循环肿瘤DNA | 肿瘤细胞凋亡 | 癌症检测 |
| 循环肿瘤细胞 | 肿瘤脱落 | 癌症分期 |
| 外泌体 | 各种细胞 | 多病种 |
| 蛋白质 | 肿瘤分泌 | 早筛 |

### 检测技术

- **NGS测序**：高通量分析
- **数字PCR**：高灵敏度检测
- **质谱分析**：蛋白质检测

## 临床应用

### 癌症领域

| 应用 | 说明 |
|------|------|
| 早筛 | 无症状人群筛查 |
| 诊断 | 辅助影像学诊断 |
| 分期 | 评估肿瘤负荷 |
| 监测 | 治疗效果评估 |
| 耐药 | 检测耐药突变 |

### 其他应用

- **产前诊断**：无创产前检测
- **器官移植**：排斥监测
- **神经疾病**：阿尔茨海默病早筛

## 技术优势

| 优势 | 说明 |
|------|------|
| 无创 | 只需抽血 |
| 实时 | 反映肿瘤动态 |
| 全面 | 克服异质性 |
| 灵敏 | 检测微量突变 |

## 挑战与局限

- **灵敏度**：早期肿瘤检出率低
- **特异性**：假阳性问题
- **标准化**：缺乏统一标准
- **成本**：检测费用较高
