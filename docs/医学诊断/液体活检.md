# 液体活检技术

## 什么是液体活检？

液体活检是通过检测血液、尿液、脑脊液等体液中的生物标志物来诊断疾病的技术。与传统组织活检相比，它具有微创、可重复采样、可动态监测等优点。

## 液体活检的主要标志物

| 标志物 | 来源 | 应用 |
|--------|------|------|
| 循环肿瘤DNA (ctDNA) | 肿瘤细胞凋亡 | 癌症早筛、疗效监测 |
| 循环肿瘤细胞 (CTC) | 肿瘤细胞脱落 | 癌症分期、预后评估 |
| 外泌体 (Exosome) | 多种细胞 | 多种疾病诊断 |
| 微小RNA (miRNA) | 多种细胞 | 疾病标志物 |

## 应用领域

### 1. 癌症早筛

液体活检可在癌症早期检测到肿瘤信号：

- **早期发现**：比影像学更早发现肿瘤
- **高危人群筛查**：适用于癌症高危人群
- **动态监测**：跟踪肿瘤发展

### 2. 疗效监测

通过 ctDNA 动态变化评估治疗效果：

- 评估治疗是否有效
- 早期发现耐药
- 指导后续治疗方案

### 3. 复发监测

术后定期检测，监测肿瘤复发：

- 比影像学更早发现复发
- 及时调整治疗策略

## 技术进展

### 单细胞测序

单细胞液体活检技术可分析单个细胞的特征。

### 多组学整合

整合基因组、表观基因组、蛋白质组数据。
