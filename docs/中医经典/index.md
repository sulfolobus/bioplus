# 中医经典

中医经典是中华民族的宝贵文化遗产，承载着数千年来先人对生命、健康和疾病的深刻认识。本板块系统介绍中医经典著作，带您领略传统医学的智慧。

## 经典著作

### 黄帝内经

中国最早的医学典籍，分为《素问》和《灵枢》两部分奠定了中医学的理论基础。

- **成书时间**：战国至西汉
- **核心思想**：阴阳五行、脏腑经络、病因病机
- **主要贡献**：确立了中医学的理论体系

### 伤寒论

东汉张仲景所著，被誉为"方书之祖"，创立了六经辨证体系。

- **成书时间**：东汉末年
- **核心思想**：六经辨证、方证对应
- **主要贡献**：奠定了中医临床医学的基础

### 金匮要略

张仲景著作的另一经典，重点论述内科杂病的诊治。

- **成书时间**：东汉末年
- **核心思想**：辨证论治、杂病诊治
- **主要贡献**：开创了中医内科杂病学

### 本草纲目

明代李时珍所著，集中国本草学之大成。

- **成书时间**：明代
- **核心思想**：药物分类、性味归经
- **主要贡献**：收载药物1892种，为世界药物学重要文献

### 养生智慧

总结中医养生理论和实践方法，涵盖饮食、起居、情志、运动等方面。

- **核心思想**：治未病、天人合一
- **主要贡献**：传承中华养生文化
