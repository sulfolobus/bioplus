# LNP技术

脂质纳米颗粒（LNP）是目前最成功的非病毒基因递送系统之一，广泛应用于mRNA疫苗和基因治疗。

## LNP 组成

### 四组分系统

| 成分 | 比例 | 功能 |
|------|------|------|
| 阳离子脂质 | 50% | 结合核酸，促进内体逃逸 |
| 磷脂 | 10% | 形成双分子层结构 |
| 胆固醇 | 38.5% | 稳定颗粒完整性 |
| PEG脂质 | 1.5% | 延长血液循环时间 |

## 作用机制

### 细胞摄取过程

1. **血浆蛋白结合**： ApoE 介导肝细胞摄取
2. **受体介导内吞**： LDLR 介导
3. **内体成熟**： pH 降低
4. **内体逃逸**： 阳离子脂质质子化
5. **核酸释放**： 进入细胞质

### 质子海绵效应

阳离子脂质在酸性环境下质子化，产生质子海绵效应，导致内体破裂，释放内容物。

## 临床应用

### mRNA 疫苗

LNP-mRNA 疫苗在 COVID-19 防控中发挥关键作用：

**成功案例：**
- Pfizer-BioNTech（Comirnaty）
- Moderna（Spikevax）

**优势：**
- 高效转染
- 安全性良好
- 可规模化生产

### 基因治疗

| 疗法 | 适应症 | 阶段 |
|------|--------|------|
| LNP-CRISPR | 肝脏疾病 | I 期 |
| siRNA-LNP | 肝脏代谢疾病 | 批准上市 |

## 优化策略

### 组织特异性

- 改变脂质组成
- 添加靶向配体
- 调节粒径

### 减少免疫激活

- 优化 PEG 含量
- 使用可降解脂质
