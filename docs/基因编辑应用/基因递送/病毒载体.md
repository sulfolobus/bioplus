# 病毒载体基因递送

病毒载体是基因治疗中最常用的递送工具，具有高转导效率和长期表达的特点。

## 常用病毒载体比较

### 腺相关病毒（AAV）

| 特点 | 说明 |
|------|------|
| 包装容量 | 4.7 kb |
| 表达时间 | 长期（可达数年） |
| 免疫原性 | 低 |
| 安全性 | 高 |

**血清型**：AAV1-AAV9（不同组织嗜性）

### 腺病毒

| 特点 | 说明 |
|------|------|
| 包装容量 | 7-36 kb |
| 表达时间 | 短期（1-2周） |
| 免疫原性 | 中等 |
| 安全性 | 较高 |

**应用**：癌症疫苗、基因治疗

### 逆转录病毒/慢病毒

| 特点 | 说明 |
|------|------|
| 包装容量 | 8 kb |
| 表达时间 | 长期（整合基因组） |
| 免疫原性 | 低 |
| 安全性 | 需关注插入突变 |

**应用**：ex vivo 基因治疗

## 组织特异性递送

### AAV 血清型选择

| 组织 | 推荐血清型 |
|------|------------|
| 肝脏 | AAV8, AAV9 |
| 肌肉 | AAV1, AAV9 |
| 心脏 | AAV9 |
| 脑 | AAV9, AAVrh10 |
| 眼睛 | AAV2, AAV5 |
| 肺部 | AAV6 |

## 挑战与优化

### 免疫原性

- 预存抗体影响疗效
- 需要免疫抑制

### 包装容量限制

- 大基因需要截短
- 多个载体系统开发
