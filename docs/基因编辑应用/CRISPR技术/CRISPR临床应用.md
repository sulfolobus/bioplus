# CRISPR 临床应用进展

CRISPR 基因编辑技术正在从实验室走向临床，本文介绍其在医学治疗中的最新进展。

## 已获批的 CRISPR 疗法

### 2023年：Casgevy 获批

**里程碑**：全球首个获批的 CRISPR 基因编辑疗法

| 项目 | 内容 |
|------|------|
| 商品名 | Casgevy |
| 通用名 | Exagamglogene autotemcel |
| 适应症 | 镰状细胞贫血 |
| 开发商 | Vertex Pharmaceuticals / CRISPR Therapeutics |
| 获批时间 | 2023年12月 |

### 2024年：扩展适应症

Casgevy 随后获批用于 **β-地中海贫血** 治疗。

## 临床试验进展

### 镰状细胞贫血

**临床数据：**
- 有效率：约 94% 患者至少 12 个月无疼痛危象
- 持久性：随访 5 年以上仍维持疗效
- 安全性：不良事件可控

### β-地中海贫血

**临床数据：**
- 输血依赖显著减少
- 约 90% 患者脱离输血依赖
- 生活质量明显改善

## 治疗流程

### 患者筛选

1. 基因诊断确认突变类型
2. 评估器官功能
3. 排除禁忌症

### 治疗步骤

```
干细胞采集 → 体外基因编辑 → 预处理化疗 → 回输 → 监护 → 随访
```

**时间周期**：约 2-3 个月

## 挑战与展望

### 当前挑战

- **高昂费用**：每次治疗费用约 200-300 万美元
- **可及性**：仅在少数医疗中心开展
- **长期安全性**：仍需更多随访数据

### 未来展望

- 降低治疗成本
- 开发体内递送系统
- 扩展适应症范围
