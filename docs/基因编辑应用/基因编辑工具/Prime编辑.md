# Prime Editing

Prime editing 是最新的基因编辑技术，能够实现所有类型的基因突变，且几乎不产生双链断裂。

## 技术原理

### 与 CRISPR-Cas9 的区别

| 方面 | Prime Editing | CRISPR-Cas9 |
|------|----------------|-------------|
| 切割方式 | 单链 nick | 双链断裂 |
| 脱靶风险 | 极低 | 存在 |
| 突变类型 | 所有类型 | 随机插入缺失 |

### 核心组件

1. **Cas9 nickase**：单链切割
2. **逆转录酶（RT）**：复制遗传信息
3. **pegRNA**：引导 + 模板

## 编辑机制

### 过程步骤

```
1. pegRNA 引导 Cas9 nickase 至目标位点
2. Cas9 切割非目标链，暴露 3' 端
3. 逆转录酶以 pegRNA 为模板，延伸 DNA
4. 细胞修复机制整合编辑序列
5. 目标链被正常链替换，完成编辑
```

### 可实现的编辑类型

| 编辑类型 | 示例 |
|----------|------|
| 插入 | 碱基插入 |
| 删除 | 碱基缺失 |
| 替换 | 点突变 |
| 组合 | 多种同时编辑 |

## 应用优势

### 精准性

- 精确到单碱基水平
- 无随机 indel
- 减少脱靶效应

### 灵活性

- 不依赖 HDR 模板
- 不需要特定序列
- 几乎所有位点可编辑

## 临床前景
