# 遗传病治疗

遗传病是由基因突变引起的疾病，基因治疗为根治这些疾病带来了希望。

## 常见遗传病类型

### 单基因遗传病

| 疾病 | 致病基因 | 发病率 |
|------|----------|--------|
| 镰状细胞贫血 | HBB | 1/500 非洲裔 |
| 囊性纤维化 | CFTR | 1/3500 高加索 |
| 杜氏肌营养不良 | DMD | 1/3500 男婴 |
| 血友病 A | F8 | 1/5000 男婴 |

### 多基因遗传病

- 2型糖尿病
- 高血压
- 阿尔茨海默病

## 基因治疗策略

### 体外基因治疗

```
患者细胞 → 基因编辑 → 体外培养 → 回输患者
```

**代表疗法**：CAR-T、造血干细胞治疗

### 体内基因治疗

```
载体 → 静脉注射/局部注射 → 靶向组织 → 基因表达
```

**代表疗法**：AAV 载体基因治疗

## 临床进展

### 已获批疗法

| 疗法 | 适应症 | 疗效 |
