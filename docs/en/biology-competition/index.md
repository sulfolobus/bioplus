## Learning Path

### Stage 1: Build the base

Start with cell biology, biochemistry, and molecular genetics.

### Stage 2: Build the system

Then move to animal physiology, plant physiology, and ecology to connect individual, population, and ecosystem thinking.

### Stage 3: Train for top scores

Finish with evolution, biotechnology, experimental methods, and data analysis for integrated questions and essays.

## Training Advice

1. Build a concept card, a question set, and an error log for each chapter.
2. Solve at least one graph question and one experimental-design question each week.
3. Use the structure: conclusion, evidence, mechanism, and boundary in every written answer.
