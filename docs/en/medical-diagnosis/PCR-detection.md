# PCR Detection Technology

Polymerase Chain Reaction (PCR) is a fundamental technology in molecular diagnostics, used to detect pathogens and gene mutations.

## Technology Principles

### Basic Principle

```
Denaturation → Annealing → Extension → Cycle
```

**Key components:**
- DNA polymerase
- Primers
- dNTPs
- Template DNA

## Technology Types

| Type | Characteristics | Application |
|------|-----------------|-------------|
| Conventional PCR | Qualitative | Gene detection |
| qPCR | Quantitative | Viral load |
| ddPCR | High-precision absolute quantification | Rare mutations |

## Clinical Applications

### Infectious Diseases

| Disease | Detection Target |
|---------|-----------------|
| COVID-19 | SARS-CoV-2 RNA |
| Hepatitis B | HBV DNA |
| Tuberculosis | TB DNA |

### Cancer Diagnosis

| Application | Description |
|-------------|-------------|
| Gene mutation detection | EGFR, KRAS |
| Fusion gene detection | ALK, ROS1 |
| Copy number variation | HER2 |
