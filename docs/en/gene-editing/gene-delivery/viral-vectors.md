# Viral Vector Gene Delivery

Viral vectors are the most commonly used delivery tools in gene therapy, offering high transduction efficiency and long-term expression.

## Comparison of Common Viral Vectors

### Adeno-Associated Virus (AAV)

| Characteristic | Description |
|----------------|-------------|
| Packaging capacity | 4.7 kb |
| Expression duration | Long-term (up to several years) |
| Immunogenicity | Low |
| Safety | High |

**Serotypes**: AAV1-AAV9 (different tissue tropisms)

### Adenovirus

| Characteristic | Description |
|----------------|-------------|
| Packaging capacity | 7-36 kb |
| Expression duration | Short-term (1-2 weeks) |
| Immunogenicity | Moderate |
| Safety | Relatively high |

**Applications**: Cancer vaccines, gene therapy

### Retrovirus/Lentivirus

| Characteristic | Description |
|----------------|-------------|
| Packaging capacity | 8 kb |
| Expression duration | Long-term (genome integration) |
| Immunogenicity | Low |
| Safety | Insertional mutagenesis concerns |

**Applications**: Ex vivo gene therapy

## Tissue-Specific Delivery

### AAV Serotype Selection

| Tissue | Recommended Serotype |
|--------|---------------------|
| Liver | AAV8, AAV9 |
| Muscle | AAV1, AAV9 |
| Heart | AAV9 |
| Brain | AAV9, AAVrh10 |
| Eye | AAV2, AAV5 |
| Lungs | AAV6 |

## Challenges and Optimization

### Immunogenicity

- Preexisting antibodies affect efficacy
- Immunosuppression required

### Packaging Capacity Limitations

- Large genes need truncation
- Multiple vector system development
