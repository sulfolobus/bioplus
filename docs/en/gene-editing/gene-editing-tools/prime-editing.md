# Prime Editing

Prime editing is the latest gene editing technology, capable of achieving all types of gene mutations with minimal double-strand breaks.

## Technology Principles

### Differences from CRISPR-Cas9

| Aspect | Prime Editing | CRISPR-Cas9 |
|--------|---------------|-------------|
| Cleavage method | Single-strand nick | Double-strand break |
| Off-target risk | Extremely low | Present |
| Mutation types | All types | Random indels |

### Core Components

1. **Cas9 nickase**: Single-strand cleavage
2. **Reverse transcriptase (RT)**: Copies genetic information
3. **pegRNA**: Guide + template

## Editing Mechanism

### Process Steps

```
1. pegRNA guides Cas9 nickase to target site
2. Cas9 cleaves non-target strand, exposing 3' end
3. Reverse transcriptase extends DNA using pegRNA as template
4. Cell repair mechanism integrates edited sequence
5. Target strand is replaced by normal strand, completing editing
```

### Editable Mutation Types

| Edit Type | Example |
|-----------|---------|
| Insertion | Base insertion |
| Deletion | Base deletion |
| Substitution | Point mutation |
| Combination | Multiple edits simultaneously |

## Application Advantages

### Precision

- Single-base level accuracy
- No random indels
- Reduced off-target effects

### Flexibility

- Does not rely on HDR template
- No specific sequence requirements
- Almost any site editable
