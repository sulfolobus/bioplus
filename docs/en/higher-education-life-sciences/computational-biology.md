# Computational Biology

## Position in the Curriculum

Builds predictive models of biological systems using mathematics, simulation, and machine learning.

## Chapter Structure and Checklist

1. Dynamical models and parameter estimation
2. Biological networks and robustness
3. Machine learning for biological data
4. Multi-omics integration
5. Prediction-experiment feedback loop
