# How Does the Immune System Work?

The immune system is the body's defense system against pathogens, protecting us from viruses, bacteria, and other harmful substances.

## Components of the Immune System

### Immune Organs

| Organ | Function |
|-------|----------|
| Bone Marrow | Produces blood cells and immune cells |
| Thymus | T cell maturation |
| Lymph Nodes | Filter lymph, immune cell aggregation |
| Spleen | Filter blood, store immune cells |
| Tonsils | First defense at respiratory entry |

### Immune Cells

- **White Blood Cells**: Main executors of the immune system
- **Macrophages**: Engulf pathogens
- **Dendritic Cells**: Present antigens to T cells
- **T Cells**: Cellular immunity
- **B Cells**: Humoral immunity, produce antibodies
- **Natural Killer Cells**: Directly kill infected cells

## Immune Mechanisms

### First Line of Defense: Innate Immunity

1. **Physical Barriers**
   - Skin
   - Mucous membranes
   - Respiratory cilia

2. **Chemical Barriers**
   - Stomach acid
   - Enzymes
   - Antimicrobial peptides

3. **Cellular Defense**
   - Macrophage engulfment
   - Natural killer cells
   - Inflammatory response

### Second Line of Defense: Adaptive Immunity

#### T Cell Immunity

- **Helper T Cells**: Coordinate immune responses
- **Cytotoxic T Cells**: Directly kill infected cells
- **Memory T Cells**: Remember pathogens

#### B Cell Immunity

- B cells produce antibodies
- Antibodies neutralize pathogens
- Memory B cells persist long-term

## Immune Response Process

```
Pathogen invasion → Antigen presentation → T cell activation → B cell antibody production → Antibody neutralization → Memory cell formation
```

## Immune System Diseases

### Immunodeficiency

- **Primary**: Genetic factors, such as SCID
- **Secondary**: HIV infection causing AIDS

### Autoimmune Diseases

The immune system mistakenly attacks own tissues:

| Disease | Target |
|---------|--------|
| Rheumatoid Arthritis | Joints |
| Systemic Lupus Erythematosus | Multiple organs |
| Type 1 Diabetes | Islet cells |
| Multiple Sclerosis | Myelin sheath |

### Allergies

Overreaction to harmless substances:

- Pollen allergies
- Food allergies
- Drug allergies

## Ways to Boost Immunity

### Healthy Lifestyle

- Balanced diet
- Moderate exercise
- Adequate sleep
- Reduce stress

### Vaccination

Vaccines help the immune system "remember" pathogens, providing long-term protection.
