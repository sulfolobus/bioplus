# Amnesia Research Progress

Amnesia refers to impaired memory function, which can be caused by various reasons and affects the storage and retrieval of information.

## Types of Amnesia

### Retrograde Amnesia

- Inability to recall events before the onset of illness
- Recent memory more affected
- Common in brain trauma, cerebrovascular disease

### Anterograde Amnesia

- Inability to form new long-term memories
- Short-term memory relatively preserved
- Common in hippocampal damage

## Etiology Classification

### Organic Causes

| Cause | Characteristics |
|-------|-----------------|
| Alzheimer's Disease | Progressive memory decline |
| Cerebrovascular Disease | Sudden onset, focal symptoms |
| Brain Trauma | Memory loss after injury |
| Encephalitis | Appears after infection |
| Epilepsy | Memory impairment during seizures |

### Functional Causes

- Psychological trauma (psychogenic amnesia)
- Depression (pseudo-dementia)
- Anxiety
- Sleep disorders

## Diagnosis

### Neuropsychological Assessment

- Memory scales (MMSE, MoCA)
- Episodic memory tests
- Working memory tests
- Language memory tests

### Imaging Studies

- MRI: Exclude structural lesions
- CT: Acute hemorrhage
- PET: Metabolic abnormalities

### Laboratory Tests

- Vitamin B12 levels
- Thyroid function
- Infection indicators

## Treatment Progress

### Drug Therapy

| Drug | Indication | Function |
|------|------------|----------|
| Donepezil | AD-related | Improve cognition |
| Memantine | Moderate-severe AD | Neuroprotection |
| Nimodipine | Vascular | Improve blood flow |
| Ginkgo Biloba | Non-specific | Improve microcirculation |

### Rehabilitation Training

#### Memory Training

- Image memory method
- Association method
- Location method
- Repetition reinforcement

#### Cognitive Training

- Computer cognitive training
- Reality orientation training
- Social interaction

### Non-pharmacological Interventions

- Lifestyle adjustment
- Sleep management
- Stress management
- Nutritional support

## Prevention Strategies

### Modifiable Risk Factors

| Factor | Prevention |
|--------|------------|
| Alcohol abuse | Limit alcohol consumption |
| Head injury | Wear seatbelt/helmet |
| Depression | Treat promptly |
| Sleep deprivation | Ensure 7-8 hours sleep |

### Protective Factors

- Lifelong learning
- Physical exercise
- Social activities
- Healthy diet (Mediterranean diet)
- Control cardiovascular risks

## Special Types

### Age-related Forgetting

- Normal memory decline associated with aging
- Daily functions preserved
- Slow progression

### Mild Cognitive Impairment (MCI)

- Between normal aging and dementia
- 10-15% progress to dementia annually
- Critical stage for intervention

### Post-traumatic Amnesia

- Caused by brain injury
- Recovery degree related to injury severity
- Early rehabilitation training important
