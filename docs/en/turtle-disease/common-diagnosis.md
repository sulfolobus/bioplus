# Common Turtle Disease Diagnosis

## Visual Diagnosis

### Healthy Turtle Characteristics

- Shell intact, hard and glossy
- Eyes bright, no discharge
- Nostrils clear, no discharge
- Strong limbs, normal crawling
- Good appetite, normal defecation

### Sick Turtle Characteristics

| Symptom | Possible Disease |
|---------|-----------------|
| Soft shell | Soft shell disease |
| Shell rot | Shell rot disease |
| Swollen eyes | Eye infection |
| Open-mouth breathing | Pneumonia |
| Weak limbs | Malnutrition/Infection |
| Refusing food | Multiple possible diseases |

## Common Disease Diagnosis

### Pneumonia

**Symptoms**:
- Open-mouth breathing
- Loud breathing sounds
- Nasal discharge
- Floating

**Diagnostic Methods**:
- Auscultation
- Observe breathing pattern
- Float test

### Gastroenteritis

**Symptoms**:
- Loss of appetite
- Abnormal stool (diarrhea/green stool)
- Foul-smelling stool
- Red anus

**Diagnostic Methods**:
- Fecal examination
- Abdominal palpation

### Soft Shell Disease

**Symptoms**:
- Soft shell
- Shell deformation
- Slow growth

**Diagnostic Methods**:
- Press shell with fingers
- Observe shell hardness

### Parasitic Infection

**Symptoms**:
- Weight loss
- Loss of appetite
- Visible parasites in stool

**Diagnostic Methods**:
- Microscopic fecal examination
- Visual parasite observation

## Home Preliminary Diagnosis

### Step 1: Observe Appearance

1. Check shell for damage, softening, spots
2. Check eyes for brightness and discharge
3. Check nostrils for blockage
4. Check skin for sores

### Step 2: Observe Behavior

1. Is it crawling normally?
2. Is appetite good?
3. Is it basking actively?
4. Any abnormal behavior?

### Step 3: Check Excretion

1. Is stool formed?
2. Is stool color normal?
3. Are there parasites?

## Veterinary Advice

### When to See a Vet

- Refusing food for more than 3 days
- Obvious behavioral abnormalities
- Severe shell lesions
- Difficulty breathing

### Preparing for Vet Visit

- Record symptoms
- Record housing environment
- Record diet
- Take photos/videos of abnormalities
