# 抗体药物原理

抗体药物是现代生物制药的核心领域，通过特异性结合靶点治疗疾病。

## 抗体结构

### Y形结构

| 区域 | 功能 |
|------|------|
| Fab区 | 抗原结合 |
| Fc区 | 效应功能 |

### 抗体类型

| 类型 | 半衰期 | 应用 |
|------|--------|------|
| IgG | 21天 | 治疗为主 |
| IgM | 5天 | 免疫诊断 |
| IgA | 6天 | 黏膜免疫 |

## 抗体药物类型

### 1. 单克隆抗体

| 特点 | 说明 |
|------|------|
| 特异性强 | 单一靶点 |
| 一致性高 | 批间差异小 |
| 半衰期长 | 给药间隔长 |

### 2. 双特异性抗体

同时靶向两个抗原，增强治疗效果。

### 3. 抗体偶联药物（ADC）

抗体+细胞毒素，实现精准化疗。

## 制备技术

### 杂交瘤技术

- 免疫动物
- 细胞融合
- 筛选阳性克隆

### 噬菌体展示

- 库容量大
- 筛选效率高

### 单B细胞克隆

- 直接从免疫动物获取
- 保留天然配对
