# 合成生物学原理

合成生物学是设计和构建新生物系统的学科，属于前沿交叉领域。

## 研究内容

### 基因线路设计

- 逻辑门基因电路
- 基因振荡器
- 细胞状态切换

### 标准化元件

| 元件类型 | 功能 |
|----------|------|
| 启动子 | 基因表达调控 |
| 终止子 | 转录终止 |
| RBS | 翻译起始调控 |

## 应用领域

### 工业生物技术

- 生物制造
- 绿色化工
- 生物燃料

### 医学应用

- 活细胞药物
- 诊断传感器
- 基因治疗载体
