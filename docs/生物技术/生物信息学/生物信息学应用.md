# 生物信息学应用

生物信息学是生命科学与信息技术的交叉学科，用于分析和解读生物数据。

## 研究方向

### 基因组学

| 应用 | 说明 |
|------|------|
| 基因组组装 | 从头测序 |
| 基因注释 | 功能预测 |
| 比较基因组 | 进化分析 |

### 蛋白质组学

- 蛋白质结构预测
- 蛋白互作网络
- 药物靶点发现

### 单细胞分析

- 细胞类型鉴定
- 发育轨迹分析
- 空间转录组

## 技术工具

| 软件/数据库 | 应用 |
|-------------|------|
| BLAST | 序列比对 |
| AlphaFold | 蛋白质结构预测 |
| UCSC Genome Browser | 基因组可视化 |
