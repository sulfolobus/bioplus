# 细胞治疗技术

细胞治疗是利用活细胞治疗疾病的前沿技术，包括干细胞治疗和免疫细胞治疗。

## 干细胞治疗

### 类型

| 干细胞类型 | 特点 |
|------------|------|
| 胚胎干细胞 | 多能性最强 |
| 诱导多能干细胞 | 避免伦理问题 |
| 成体干细胞 | 安全性高 |

### 临床应用

- 血液系统疾病
- 神经系统疾病
- 软骨修复
