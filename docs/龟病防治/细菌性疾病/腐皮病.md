# 腐皮病

腐皮病是龟类常见的皮肤疾病，主要由细菌感染引起，表现为皮肤溃烂、坏死。该病在水质不洁或龟体受伤时容易发生。

## 病因

### 病原学

- 气单胞菌（Aeromonas spp.）
- 假单胞菌（Pseudomonas spp.）
- 变形杆菌（Proteus spp.）
- 金黄色葡萄球菌（Staphylococcus aureus）

### 诱因

- 水质恶化
- 皮肤外伤（摩擦、咬伤）
- 免疫力下降
- 饲养密度过高
- 温度不适

## 症状

### 早期症状

- 皮肤局部发白
- 皮肤软化
- 患处皮肤轻微肿胀
- 龟食欲正常或稍减

### 中期症状

- 皮肤溃烂扩大
- 溃烂边缘不规则
- 有脓性分泌物
- 患处发出异味
- 食欲下降

### 晚期症状

- 大面积皮肤坏死
- 皮肤脱落
- 肌肉暴露
- 继发全身感染
- 精神萎靡、拒食

## 诊断

### 临床诊断

- 观察皮肤病变
- 触摸患处
- 检查病变范围和深度

### 实验室诊断

- 分泌物涂片检查
- 细菌培养

## 治疗

### 局部治疗

1. 清洁患处：生理盐水冲洗
2. 消毒：碘伏或高锰酸钾溶液
3. 上药：抗生素软膏
4. 干养：每天定时干养

### 全身治疗

| 药物 | 剂量 | 用法 | 疗程 |
|------|------|------|------|
| 恩诺沙星 | 10mg/kg | 药浴 | 7天 |
| 土霉素 | 40mg/kg | 口服 | 7天 |
| 头孢氨苄 | 20mg/kg | 口服 | 7天 |

### 环境调整

- 改善水质
- 降低密度
- 升温至30℃

## 预防

- 保持水质清洁
- 定期换水
- 避免皮肤外伤
- 合理密度
- 营养均衡
