# 蜱虫病

蜱虫病是由蜱虫寄生引起的体外寄生虫病。蜱虫不仅造成皮肤损伤，还可传播血液寄生虫，是龟类健康的重要威胁。

## 病因

### 病原学

- 蜱螨目（Ixodida）
- 硬蜱科（Ixodidae）
- 软蜱科（Argasidae）

### 寄生部位

- 颈部皮肤
- 四肢基部
- 腋下
- 皮肤褶皱处

## 症状

### 早期症状

- 皮肤可见蜱虫附着
- 轻微瘙痒
- 龟不安、蹭抓
- 食欲正常

### 中期症状

- 蜱虫数量增多
- 皮肤红肿
- 贫血（大量寄生时）
- 消瘦

### 晚期症状

- 严重贫血
- 皮肤继发感染
- 蜱媒疾病传播
- 死亡

## 诊断

- 肉眼检查虫体
- 虫体形态鉴定

## 治疗

### 人工去除

1. 用镊子夹住蜱虫
2. 缓慢旋转拔出
3. 避免口器断裂
4. 消毒叮咬部位

### 药物治疗

| 药物 | 用法 |
|------|------|
| 伊维菌素 | 外用/口服 |
| 塞拉菌素 | 外用滴剂 |
| 非泼罗尼 | 外用滴剂 |

### 环境处理

- 驱杀环境中的蜱虫
- 彻底消毒

## 预防

- 定期检查
- 避免野外放养
- 环境驱虫
- 新龟隔离检查
