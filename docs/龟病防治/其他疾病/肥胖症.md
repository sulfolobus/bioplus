# 肥胖症

肥胖症是龟类常见的营养代谢疾病，由于长期过量喂食或运动不足导致脂肪过度堆积。肥胖会影响龟的健康，缩短寿命。

## 病因

### 营养过剩

- 长期过量喂食
- 高脂肪食物
- 蛋白质摄入过多
- 碳水化合物过量

### 运动不足

- 饲养空间过小
- 缺乏活动
- 长期室内饲养

### 其他因素

- 代谢紊乱
- 内分泌失调
- 年龄（老年龟更易发胖）

## 症状

### 早期症状

- 四肢根部肥胖
- 颈部肥胖
- 体重增加
- 活动正常

### 中期症状

- 明显肥胖外观
- 行动迟缓
- 爬行困难
- 食欲旺盛

### 晚期症状

- 严重肥胖
- 脂肪肝
- 心脏负担加重
- 行动困难
- 寿命缩短

## 诊断

- 外观检查
- 称重对比
- 触诊检查脂肪堆积

## 治疗

### 饮食控制

1. **减少喂食量**
   - 逐步减少喂食量
   - 成年龟每周喂食2-3次

2. **调整饮食结构**
   - 减少高脂食物
   - 增加蔬果比例
   - 选择低脂龟粮

### 增加运动

- 扩大饲养空间
- 设置障碍物
- 诱导活动

### 环境调整

- 降低温度（促进活动）
- 增加光照时间

## 预防

- 适量喂食
- 合理饮食结构
- 足够活动空间
- 定期称重监测
