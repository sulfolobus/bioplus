# 软壳病

软壳病是龟类常见的代谢性疾病，主要由于钙质或维生素D缺乏引起，表现为甲壳软化、变形。该病影响甲壳的硬度，严重时龟无法正常活动。

## 病因

### 营养缺乏

- 钙质摄入不足
- 维生素D3缺乏
- 磷钙比例失调
- 蛋白质缺乏

### 环境因素

- 缺乏紫外线照射
- 温度不适
- 湿度不当

### 其他因素

- 吸收障碍
- 肝脏疾病
- 甲状腺功能异常
- 长期患病

## 症状

### 早期症状

- 甲壳轻微变软
- 甲壳颜色变浅
- 龟仍能正常活动
- 食欲正常

### 中期症状

- 甲壳明显软化
- 按压有弹性感
- 甲壳边缘向上翘起
- 生长停滞

### 晚期症状

- 严重软化、变形
- 甲壳塌陷
- 运动困难
- 食欲下降
- 继发感染

## 诊断

- 触摸甲壳硬度
- 观察甲壳形态
- 血液检查（血钙、血磷）
- X光检查

## 治疗

### 营养补充

| 药物 | 剂量 | 用法 |
|------|------|------|
| 葡萄糖酸钙 | 50-100mg/kg | 口服/注射 |
| 碳酸钙 | 100-200mg/kg | 口服 |
| 维生素D3 | 1000-2000IU/kg | 口服/注射 |

### 环境调整

- 增加紫外线照射（每天1-2小时）
- 保持适宜温度（28-30℃）
- 设置晒台

### 饮食调整

- 添加墨鱼骨
- 喂食小鱼小虾
- 补充钙粉
- 均衡饮食

## 预防

- 日常补充钙质
- 充足紫外线照射
- 均衡饮食
- 定期检查甲壳硬度
