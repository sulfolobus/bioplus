# 膀胱结石

膀胱结石是龟类常见的泌尿系统疾病，由于矿物质代谢异常或脱水导致尿液中晶体沉积形成结石。结石可引起排尿困难、腹痛等症状。

## 病因

### 代谢因素

- 矿物质代谢异常
- 钙磷比例失调
- 尿酸盐沉积

### 诱因

- 脱水
- 长期低钙饮食
- 维生素A缺乏
- 尿液pH异常
- 遗传因素

## 症状

### 早期症状

- 排尿困难
- 少量多次排尿
- 食欲正常
- 活动正常

### 中期症状

- 腹痛（弓背）
- 食欲减退
- 腹部可触及硬块
- 排尿时痛苦

### 晚期症状

- 完全尿闭
- 腹部膨胀
- 精神萎靡
- 肾衰竭
- 死亡

## 诊断

- 触诊检查
- X光检查
- B超检查
- 尿液分析

## 治疗

### 保守治疗

1. **增加饮水**
   - 泡澡
   - 口服补液

2. **药物治疗**
   - 软化结石药物
   - 利尿剂

### 手术治疗

- 体外结石取出
- 手术取出
- 膀胱切开术

## 预防

- 保证充足饮水
- 适量运动
- 均衡饮食
- 定期检查
