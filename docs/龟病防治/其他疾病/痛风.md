# 痛风

痛风是龟类常见的代谢性疾病，由于尿酸盐代谢异常在关节和内脏器官沉积所致。该病引起剧烈疼痛，严重影响龟的活动能力和健康。

## 病因

### 嘌呤代谢异常

- 蛋白质摄入过多
- 核酸代谢旺盛
- 尿酸排泄障碍

### 诱因

- 高蛋白饮食
- 脱水
- 肾功能受损
- 维生素缺乏
- 药物影响

## 症状

### 关节痛风

- 关节肿胀
- 跛行
- 疼痛（不愿活动）
- 关节僵硬

### 内脏痛风

- 食欲不振
- 嗜睡
- 消瘦
- 肾区肿大

### 病理变化

- 关节白色结晶沉积
- 肾脏白色斑点
- 内脏表面白色沉积

## 诊断

- 临床症状
- X光检查
- 血液检查（尿酸升高）
- 病理剖检

## 治疗

### 药物治疗

| 药物 | 剂量 | 用法 |
|------|------|------|
| 秋水仙碱 | 0.1mg/kg | 口服 |
| 别嘌醇 | 10mg/kg | 口服 |
| 碳酸氢钠 | 100mg/kg | 口服 |

### 支持治疗

- 补液
- 止痛
- 营养支持

### 饮食调整

- 降低蛋白摄入
- 增加饮水
- 补充维生素

## 预防

- 均衡饮食
- 适量蛋白
- 保证饮水
- 定期检查
