# 蛔虫病

蛔虫病是龟类最常见的体内寄生虫病之一，由蛔虫寄生于肠道引起。轻度感染无症状，严重时可导致肠道阻塞、营养不良甚至死亡。

## 病因

### 病原学

- 蛔虫科（Ascarididae）
- 主要为Ascaridia属和Hexametra属
- 虫体呈乳白色，圆柱形

### 感染途径

- 吞食感染性虫卵
- 摄入中间宿主（蟑螂、苍蝇等）
- 污染的食物和水源

## 症状

### 轻度感染

- 无明显症状
- 食欲正常
- 粪便中可见虫卵

### 中度感染

- 食欲减退
- 消瘦
- 毛发暗淡
- 粪便中可见虫体
- 轻度贫血

### 重度感染

- 食欲不振或拒食
- 严重消瘦
- 腹部膨胀
- 肠道阻塞
- 呕吐（可能吐虫）
- 死亡

## 诊断

### 临床诊断

- 粪便检查虫卵
- 粪便中可见虫体
- 消瘦症状

### 实验室诊断

- 粪便显微镜检查
- 虫卵形态鉴定

## 治疗

### 驱虫治疗

| 药物 | 剂量 | 用法 | 重复给药 |
|------|------|------|----------|
| 阿苯达唑 | 25-50mg/kg | 口服 | 2周后重复 |
| 芬苯达唑 | 50mg/kg | 口服 | 2周后重复 |
| 吡喹酮 | 5-10mg/kg | 口服 | 2周后重复 |
| 左旋咪唑 | 10mg/kg | 口服 | 2周后重复 |

### 支持治疗

- 补充营养
- 维生素B族
- 铁剂（贫血时）
- 易消化食物

## 预防

- 定期驱虫（3-6个月一次）
- 保持环境清洁
- 食物来源干净
- 定期消毒
- 避免昆虫进入饲养环境
