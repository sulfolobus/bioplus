# 常见龟病诊断

## 外观诊断

### 健康龟特征

- 甲壳完整、坚硬有光泽
- 眼睛明亮、无分泌物
- 鼻孔通畅、无分泌物
- 四肢有力、爬行正常
- 食欲良好、排便正常

### 患病龟特征

| 症状 | 可能疾病 |
|------|----------|
| 甲壳软化 | 软壳病 |
| 甲壳腐烂 | 甲壳腐烂病 |
| 眼部肿胀 | 眼部感染 |
| 张口呼吸 | 肺炎 |
| 四肢无力 | 营养不良/感染 |
| 拒食 | 多种疾病可能 |

## 常见疾病诊断

### 肺炎

**症状**：
- 张口呼吸
- 呼吸声大
- 鼻孔有分泌物
- 浮水

**诊断方法**：
- 听诊呼吸音
- 观察呼吸方式
- 浮水测试

### 肠胃炎

**症状**：
- 食欲不振
- 排便异常（稀便/绿便）
- 粪便有异味
- 肛门红肿

**诊断方法**：
- 粪便检查
- 触摸腹部

### 软壳病

**症状**：
- 甲壳软化
- 甲壳变形
- 生长缓慢

**诊断方法**：
- 手指按压甲壳
- 观察甲壳硬度

### 寄生虫感染

**症状**：
- 消瘦
- 食欲不振
- 粪便可见虫体

**诊断方法**：
- 粪便显微镜检查
- 肉眼观察虫体

## 家庭初步诊断

### Step 1: 观察外观

1. 检查甲壳是否有破损、软化、斑点
2. 检查眼睛是否明亮、有无分泌物
3. 检查鼻孔是否通畅
4. 检查皮肤是否有溃烂

### Step 2: 观察行为

1. 是否正常爬行
2. 是否食欲良好
3. 是否主动晒太阳
4. 是否有异常行为

### Step 3: 检查排泄

1. 粪便是否成形
2. 粪便颜色是否正常
3. 是否有寄生虫

## 就医建议

### 何时就医

- 拒食超过 3 天
- 明显行为异常
- 甲壳严重病变
- 呼吸困难

### 就诊准备

- 记录症状
- 记录饲养环境
- 记录饮食情况
- 拍照/视频记录异常
