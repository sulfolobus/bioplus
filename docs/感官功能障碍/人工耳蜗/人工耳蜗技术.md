# 人工耳蜗技术

人工耳蜗是帮助重度听力损失患者恢复听觉的电子装置。

## 工作原理

### 组成部件

| 部件 | 位置 | 功能 |
|------|------|------|
| 麦克风 | 体外 | 收集声音 |
| 言语处理器 | 体外 | 处理声音信号 |
| 刺激线圈 | 体内 | 传递信号 |
| 电极阵列 | 耳蜗内 | 刺激听神经 |

### 工作流程

```
声音 → 麦克风 → 言语处理器 → 编码 → 刺激线圈 → 电极 → 听神经 → 大脑
```

## 临床应用

### 适应人群

- 双耳重度至极重度感音神经性听力损失
- 助听器效果不佳
- 无手术禁忌症

### 植入效果

| 效果 | 说明 |
|------|------|
| 声觉康复 | 感知声音 |
| 言语理解 | 理解语言 |
| 音乐欣赏 | 部分患者可欣赏音乐 |

## 技术发展

- 软电极植入
- 残余听力保护
- 双侧植入
- 声电联合刺激
