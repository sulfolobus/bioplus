# 听力损失治疗

听力损失是常见的感官功能障碍，本文介绍其治疗策略。

## 听力损失类型

| 类型 | 原因 | 治疗 |
|------|------|------|
| 传导性 | 外耳/中耳病变 | 手术/助听器 |
| 感音神经性 | 内耳/听觉神经 | 助听器/人工耳蜗 |
| 混合性 | 两者兼有 | 综合治疗 |

## 治疗策略

### 药物治疗

- 糖皮质激素（突发性耳聋）
- 血管扩张剂
- 营养神经药物

### 器械治疗

| 设备 | 适用情况 |
|------|----------|
| 助听器 | 轻中度听力损失 |
| 人工耳蜗 | 重度至极重度感音神经性聋 |
| 骨传导助听器 | 传导性听力损失 |
