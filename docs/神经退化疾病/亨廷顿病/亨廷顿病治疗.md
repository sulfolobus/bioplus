# 亨廷顿病治疗

亨廷顿病是一种常染色体显性遗传的神经退行性疾病。

## 疾病特征

### 病因

- HTT基因CAG重复扩增
- 突变亨廷顿蛋白产生
- 神经毒性

### 症状

| 症状类型 | 表现 |
|----------|------|
| 运动 | 不自主舞蹈样动作 |
| 认知 | 记忆力下降 |
| 精神 | 抑郁、焦虑 |

## 治疗进展

### 对症治疗

- 丁苯那嗪（舞蹈样动作）
- 抗精神病药物

### 新疗法

- 基因编辑治疗
- RNAi治疗
- 小分子抑制剂
