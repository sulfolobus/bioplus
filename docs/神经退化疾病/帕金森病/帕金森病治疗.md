# 帕金森病治疗

帕金森病是常见的神经退行性疾病，本文介绍其治疗进展。

## 疾病特征

### 病理特征

- 黑质多巴胺能神经元丢失
- Lewy小体形成
- 路易体沉积

### 症状

| 运动症状 | 非运动症状 |
|----------|------------|
| 静止性震颤 | 嗅觉减退 |
| 肌强直 | 睡眠障碍 |
| 运动迟缓 | 便秘 |
| 姿势障碍 | 抑郁 |

## 治疗策略

### 药物治疗

| 药物类型 | 代表药物 |
|----------|----------|
| 左旋多巴 | 美多芭 |
| 多巴胺受体激动剂 | 普拉克索 |
| MAO-B抑制剂 | 雷沙吉兰 |

### 外科治疗

- DBS（脑深部刺激）
- 神经核损毁术

## 新疗法进展

- 基因治疗
- 干细胞移植
- 免疫治疗
