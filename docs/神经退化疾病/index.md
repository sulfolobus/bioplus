# 神经-退化疾病

神经退行性疾病是由神经元结构或功能逐渐丧失导致的疾病，主要影响中枢和外周神经系统。

## 主要疾病类型

### 阿尔茨海默症

最常见的神经退行性疾病，主要表现为记忆减退和认知功能障碍。

### 帕金森病

影响运动控制的神经退行性疾病，主要症状包括震颤、僵硬和运动迟缓。

### 健忘症

记忆力受损，可由多种原因引起，包括 aging、创伤、疾病等。

## 研究方向

### 病因研究

- 蛋白质错误折叠与聚集
- 神经炎症
- 遗传因素
- 环境因素

### 治疗方法

- 药物治疗
- 干细胞疗法
- 基因治疗
- 生活方式干预
