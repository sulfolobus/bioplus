# 阿尔茨海默症研究进展

阿尔茨海默症（Alzheimer's Disease, AD）是最常见的神经退行性疾病，占所有痴呆病例的 60-70%。

## 疾病特征

### 病理特征

- **β-淀粉样蛋白斑块**：细胞外的异常蛋白质沉积
- **tau 蛋白纤维缠结**：细胞内的异常蛋白质结构
- **神经元死亡**：特别是海马体和皮层的神经元

### 症状表现

| 阶段 | 症状 |
|------|------|
| 早期 | 记忆力减退，尤其是近期记忆；找词困难 |
| 中期 | 认知功能下降，定向障碍，行为改变 |
| 晚期 | 严重记忆障碍，失去语言能力，生活不能自理 |

## 发病机制

### 主流理论

1. **淀粉样蛋白假说**
   - Aβ 蛋白过度产生或清除不足
   - 导致神经元毒性

2. **tau 蛋白假说**
   - tau 蛋白过度磷酸化
   - 导致神经纤维缠结

3. **神经炎症假说**
   - 慢性神经炎症加速病情进展

### 风险因素

| 因素 | 影响 |
|------|------|
| 年龄 | 最大风险因素，65 岁后风险显著增加 |
| 遗传 | APP、PSEN1、PSEN2 基因突变 |
| 生活方式 | 缺乏运动、不良饮食、吸烟 |
| 心血管疾病 | 高血压、糖尿病、高胆固醇 |

## 诊断方法

### 临床诊断

- 认知功能评估（MMSE、MoCA）
- 病史采集
- 神经系统检查

### 生物标志物

| 标志物 | 检测方法 | 意义 |
|--------|----------|------|
| Aβ42 | 脑脊液检测 | 降低提示 AD |
| tau 蛋白 | 脑脊液检测 | 升高提示 AD |
| PET 扫描 | 淀粉样蛋白成像 | 斑块沉积 |

## 治疗进展

### 已批准药物

| 药物 | 类型 | 作用 |
|------|------|------|
| 美金刚 | NMDA 拮抗剂 | 改善认知 |
| 多奈哌齐 | 胆碱酯酶抑制剂 | 改善症状 |
| Aducanumab | 抗 Aβ 抗体 | 清除斑块（争议） |
| Lecanemab | 抗 Aβ 抗体 | 减缓进展 |
| Donanemab | 抗 Aβ 抗体 | 清除斑块 |

### 在研疗法

#### 免疫疗法

- 主动免疫疫苗
- 单克隆抗体
- 纳米抗体

#### 基因治疗

- APOE 基因编辑
- 神经生长因子基因治疗

#### 干细胞疗法

- 胚胎干细胞来源的神经元
- 诱导多能干细胞
- 神经干细胞移植

### 非药物干预

- 认知训练
- 运动疗法
- 音乐疗法
- 虚拟现实训练

## 预防策略

### 可改变风险因素

| 风险因素 | 预防措施 |
|----------|----------|
| 缺乏运动 | 每周 150 分钟中等强度运动 |
| 吸烟 | 戒烟 |
| 肥胖 | 维持健康体重 |
| 高血压 | 控制血压 |
| 糖尿病 | 控制血糖 |
| 抑郁 | 及时治疗 |

### 保护因素

- 高等教育
- 社交活动
- 认知刺激
- 地中海饮食
- 适量饮酒

## 最新研究动态

### 2024-2025 年进展

- Tau 蛋白成像技术突破
- 早期筛查血液检测
- 微生物组与 AD 关系研究
- AI 在诊断中的应用

### 未来展望

- 个性化医疗
- 组合疗法
- 疾病修饰疗法
- 早期干预
