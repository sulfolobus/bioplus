# 生理学

## 学科定位

研究机体功能调控机制，是医学、药学和生命科学的核心基础课程。

## 章节与知识点清单

1. 内环境与稳态调控
2. 神经与内分泌调节
3. 循环、呼吸与能量代谢
4. 消化、肾脏与体液平衡
5. 器官系统耦合与疾病表型
