# 生物信息学

## 学科定位

通过数据库、统计与算法处理生物数据，是组学研究和精准医学关键工具。

## 章节与知识点清单

1. 生物数据库与数据标准
2. 序列比对与注释
3. 基因组组装与变异分析
4. 转录组/蛋白组分析流程
5. 可重复分析与结果可视化
