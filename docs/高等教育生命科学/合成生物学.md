# 合成生物学

## 学科定位

用工程化思路设计和构建生物系统，是生物制造和新药研发前沿方向。

## 章节与知识点清单

1. 标准生物元件与设计原则
2. 基因电路构建与调试
3. 代谢工程与底盘细胞
4. Design-Build-Test-Learn 循环
5. 生物安全与伦理治理
