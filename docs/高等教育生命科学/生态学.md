# 生态学

## 学科定位

研究生物与环境互作规律，从种群到生态系统尺度分析生命活动。

## 章节与知识点清单

1. 个体生态与生态位
2. 种群动态与生活史策略
3. 群落结构与种间关系
4. 生态系统能量流动与物质循环
5. 全球变化与生物多样性保护
