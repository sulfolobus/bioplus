# 分子生物学

## 学科定位

围绕 DNA-RNA-蛋白质信息流动研究遗传信息表达与调控机制。

## 章节与知识点清单

1. DNA 复制、损伤与修复
2. 转录与转录调控
3. 翻译与蛋白质加工
4. 表观遗传与非编码 RNA
5. 分子实验技术与功能验证
