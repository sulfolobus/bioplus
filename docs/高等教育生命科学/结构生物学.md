# 结构生物学

## 学科定位

利用三维结构信息解释生物分子功能、相互作用与药物作用机制。

## 章节与知识点清单

1. 生物大分子结构层级
2. X 射线晶体学基础
3. NMR 与冷冻电镜方法
4. 分子对接与复合体结构
5. 结构驱动药物设计
