# 病毒学

## 学科定位

研究病毒的结构、复制、进化与宿主互作，连接基础生物学、免疫学与公共卫生。

## 章节与知识点清单

1. 病毒分类与命名体系
2. 病毒结构与基因组类型
3. 病毒复制周期与宿主侵染
4. 病毒进化、变异与跨种传播
5. 病毒学检测、疫苗与抗病毒策略
