# 微生物学

## 学科定位

研究细菌、古菌、真菌、病毒等微生物的结构、代谢、遗传与应用。

## 章节与知识点清单

1. 微生物分类与鉴定
2. 微生物生长与代谢
3. 微生物遗传与抗性
4. 微生物生态与微生物组
5. 医学与工业微生物学
