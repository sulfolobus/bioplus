# 遗传学

## 学科定位

研究遗传信息的传递与变异，是现代生命科学和医学的重要理论基础。

## 章节与知识点清单

1. 孟德尔遗传与扩展规律
2. 连锁交换与遗传作图
3. 染色体与基因组结构
4. 数量遗传与群体遗传
5. 医学遗传与复杂性状
