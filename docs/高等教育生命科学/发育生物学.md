# 发育生物学

## 学科定位

研究从受精卵到成体的时空发育程序，是再生医学与先天疾病研究基础。

## 章节与知识点清单

1. 配子发生与受精
2. 胚胎早期发育与胚层形成
3. 轴建立与形态发生
4. 器官发生与细胞命运决定
5. 干细胞、再生与发育异常
