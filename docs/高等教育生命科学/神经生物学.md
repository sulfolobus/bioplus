# 神经生物学

## 学科定位

从分子、细胞、回路到行为层面研究神经系统信息处理与调控。

## 章节与知识点清单

1. 神经元电生理与离子通道
2. 突触传递与突触可塑性
3. 感觉-运动系统组织
4. 学习记忆与行为神经机制
5. 神经退行性疾病与神经修复
