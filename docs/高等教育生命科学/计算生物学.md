# 计算生物学

## 学科定位

通过数学建模、仿真和机器学习研究生命系统，强调定量预测与验证。

## 章节与知识点清单

1. 动力学模型与参数估计
2. 网络生物学与系统稳定性
3. 机器学习与生命数据分析
4. 多组学整合建模
5. 预测-实验闭环
