# 生物化学

## 学科定位

研究生命分子的结构、性质、反应与能量转换，是微观层面核心课程。

## 章节与知识点清单

1. 蛋白质、核酸、脂质、糖类结构
2. 酶学与反应动力学
3. 代谢通路与代谢调控
4. 生物能学与氧化磷酸化
5. 代谢异常与疾病机制
