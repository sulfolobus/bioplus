# 生命科学导论

## 学科定位

建立生命科学总体框架，帮助学习者理解从生态系统到分子机制的多层级联系。

## 章节与知识点清单

1. 生命的层级：个体、器官、细胞、分子、基因组
2. 科学方法：假设、实验设计、对照与统计
3. 生命科学史：经典理论与技术革命
4. 当代前沿：组学、AI、生物制造、精准医学
