# 细胞生物学

## 学科定位

以细胞为核心单位研究生命活动，连接组织层面与分子层面机制。

## 章节与知识点清单

1. 细胞结构与膜系统
2. 细胞骨架、运输与运动
3. 细胞周期与凋亡
4. 细胞信号转导网络
5. 细胞分化、衰老与肿瘤
