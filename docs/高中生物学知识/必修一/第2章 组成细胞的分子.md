# 第2章 组成细胞的分子

## 知识点清单

- 生物体内常见元素与化合物分类
- 糖类、脂质、蛋白质、核酸的结构与功能
- 水和无机盐在生命活动中的作用
- 常见有机物检测实验原理

## 典题

题目：为什么说蛋白质是生命活动的主要承担者？请从结构和功能两个角度回答。

解析思路：先写氨基酸组成与空间结构多样性，再对应酶、抗体、运输和调节等功能。

## 易错点

- 将“核酸”只理解为 DNA，忽略 RNA
- 混淆还原糖检测与脂肪检测方法
- 把“脂质都不溶于水”绝对化
