## 零基础学习路线

### 第 1 阶段：建立概念地基（2-4 周）

- 先学必修一，掌握细胞、酶、呼吸与光合作用。
- 每学完一节，完成“定义 + 原理 + 例题”的三联笔记。

### 第 2 阶段：打通遗传主线（2-3 周）

- 学必修二，把“减数分裂—遗传规律—基因表达—变异—进化”连成链。
- 每天固定做 3-5 道遗传计算题，训练审题与表达。

### 第 3 阶段：提升系统思维（2-3 周）

- 学必修三，用“稳态—调节—生态”理解复杂系统。
- 重点突破实验图表题和生态能量计算题。

### 第 4 阶段：选修拓展与综合训练（2-4 周）

- 按考试要求补选修内容，重点看实验技术、科技应用与生物伦理。
- 做跨章节综合题，训练完整论证能力。

## 高分训练方法

1. 概念过关：每个关键词要能“定义 + 举例 + 反例”。
2. 图表过关：看到曲线、柱状图、流程图要先提取变量关系。
3. 实验过关：会写对照、变量、重复和误差分析。
4. 表达过关：结论必须对应题干与数据，不写空话。

## 每周学习模板

- 周一到周三：新课学习 + 例题
- 周四：专题整理 + 错题回放
- 周五：实验与图表专题
- 周末：综合卷训练 + 复盘

## 知识覆盖清单

- 必修一：细胞分子基础、细胞结构、物质运输、酶与 ATP、呼吸与光合、细胞生命历程。
- 必修二：遗传规律、减数分裂、DNA 与基因表达、变异、育种、进化理论。
- 必修三：稳态与调节、神经体液免疫、植物激素、种群群落、生态系统、生态保护。
- 选修一：发酵、微生物培养、组织培养、酶工程实验。
- 选修二：生物技术与社会、生物伦理、公共卫生、可持续发展。
- 选修三：基因工程、细胞工程、胚胎工程、生物安全与技术评估。

使用方法：每学完一个小节，按“概念是否会解释、题目是否会做、实验是否会分析”逐项打勾。
