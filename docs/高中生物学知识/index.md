# 高中生物学知识

这里汇总高中阶段常见的生物学核心概念、实验思路和题型方法，适合做系统复习和查缺补漏。

## 学习重点

- 细胞结构与功能
- 遗传与变异
- 代谢与能量转换
- 稳态与调节
- 生态系统与环境
- 实验设计与数据分析

## 学习路径

### 1. 先建立概念框架

把教材中的关键词、图示和定义串成一张知识图，先理解再记忆。

### 2. 再整理常见题型

把选择题、实验题、图表题和综合题分开整理，形成可复用的解题步骤。

### 3. 最后回到实验与应用

结合真实情境理解生物学原理，训练从现象到结论的表达能力。
